
# Customer Support Annotation Project

## Objective
Industry-level data annotation project demonstrating NLP annotation workflows,
LLM evaluation, and dashboard visualization.

## Skills Demonstrated
- Data Annotation
- NLP Fundamentals
- LLM Evaluation
- React Dashboard Development
- Data Analysis

## Workflow
1. Create realistic customer queries dataset.
2. Annotate intents, entities, sentiment, relevance.
3. Evaluate LLM responses.
4. Visualize annotations in React dashboard.

## Dashboard Features
- Annotated sample count
- Sentiment overview
- Data table
- Modern UI (React + Tailwind)
